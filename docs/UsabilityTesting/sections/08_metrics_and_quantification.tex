\section{Metrics and Quantification}
\label{sec:metrics}

This section defines the measures used in Section~\ref{sec:results}.

\subsection{Task Performance}

\begin{itemize}
  \item \textbf{Task success rate:} For each of UT-1 through UT-7, the share of participants who met the scenario success criteria. Report per task and overall. (UT-2R was not run.)
  \item \textbf{Time on task:} Time from moderator prompt to success or stop; report median, interquartile range, and notable outliers.
  \item \textbf{Assists:} Count light hints vs.\ substantive help; optional composite ``error score'' if the team defines weights up front.
  \item \textbf{Critical incidents:} Count problems by severity (blocks task / major / minor) using a rubric fixed before sessions.
\end{itemize}

\subsection{Subjective Measures}

\begin{itemize}
  \item \textbf{SUS score:} Per participant and mean $\pm$ standard deviation across $N$.
  \item \textbf{Post-task ease (1--7):} Mean per task from PT3 (Section~\ref{sec:questions_surveys}).
  \item \textbf{Optional Likert blocks:} Means per theme (e.g.\ trust, clarity).
\end{itemize}

\subsection{Analysis}

\begin{itemize}
  \item Summarize numbers in tables; add charts if the course requires them.
  \item Tag themes in open-ended answers; link to screenshots or timestamps when recordings exist.
  \item Tie major issues to SRS FR/NFR or product areas when useful.
\end{itemize}

\subsection{Thresholds (optional targets)}

Record any targets set before testing (e.g.\ at least 80\% success on UT-1, median SUS $\geq$ 68).
